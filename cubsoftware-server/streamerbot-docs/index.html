Quick Start

1

Install ConfigSetup.cs

This is the foundation file. Run it first to initialize all settings.

2

Add Your Commands

Import any command .cs file into StreamerBot actions.

3

Set Chat Trigger

Link the action to a chat command like !daily or !slots.

4

Go Live!

Test in chat and enjoy your new currency system.

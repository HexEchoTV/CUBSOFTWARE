CubPresence Extension

Install the browser extension to connect CubPresence to Discord

Installation Guide

  1. Download the Extension

    Click the download button below to get the extension files (.zip)

  2. Extract the ZIP file

    Unzip the downloaded file to a folder on your computer

  3. Open Extension Settings

    Go to chrome://extensions (or your browser's extension page) and enable "Developer mode"

  4. Load the Extension

    Click "Load unpacked" and select the folder where you extracted the extension

  5. You're Ready!

    Go back to CubPresence and click "Connect to Discord" - the extension will handle the rest
